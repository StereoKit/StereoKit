## Description
Type: XrSpace. StereoKit's primary coordinate space,
valid after SK.Initialize, this will most likely be created
from `XR_REFERENCE_SPACE_TYPE_UNBOUNDED_MSFT` or
`XR_REFERENCE_SPACE_TYPE_LOCAL`.
